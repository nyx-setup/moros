# MOROS

> Hardware security research device — built in public.

---

The goal is not just to build a device, but to make the invisible visible: how signals work, how protocols behave, what happens at the hardware level.

## Hardware Base

| Component | Details |
|-----------|---------|
| Platform | LilyGO T-Embed CC1101 Plus |
| MCU | ESP32-S3 |
| RF | CC1101 Sub-GHz transceiver |
